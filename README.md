This repository contains a Streamlit application for discrete event simulation and queuing calculations.

Quick local run:

```bash
python -m pip install -r requirements.txt
streamlit run streamlit_app.py
```

Deploy to Streamlit Cloud (share.streamlit.io):

1. Push this repository to GitHub.
2. Go to https://share.streamlit.io and sign in with your GitHub account.
3. Click "New app" → select the repo and branch → set the main file to `streamlit_app.py` → Deploy.

Notes:
- `requirements.txt` lists the app dependencies.
- If you change package versions, update `requirements.txt` accordingly.
